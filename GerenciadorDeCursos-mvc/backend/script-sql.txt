Tabela students:
CREATE TABLE students (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(255) NOT NULL,
    email VARCHAR(255) UNIQUE NOT NULL);

Tabela courses:
CREATE TABLE courses (
    id INT AUTO_INCREMENT PRIMARY KEY,
    title VARCHAR(255) NOT NULL,
    description TEXT );

Tabela enrollments:
*(Tabela intermediária para o relacionamento muitos-para-muitos):
CREATE TABLE enrollments (
    student_id INT,
    course_id INT,
    PRIMARY KEY(student_id, course_id),
    FOREIGN KEY (student_id) REFERENCES students(id),
    FOREIGN KEY (course_id) REFERENCES courses(id));

criar aluno: POST http://localhost:3000/students { "name": "Cristiano Ronaldo", "email": "cristiano.ronaldo@exemplo.com" }

listar cursos: GET	http://localhost:3000/courses

listar cursos de um aluno: GET http://localhost:3000/students/1/courses
listar alunos de um curso: GET	http://localhost:3000/courses/2/students
